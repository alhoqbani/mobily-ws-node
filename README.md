# Mobily Ws
A Node.js client to communicate with mobily.ws API

## Installation
```sh
npm install mobily-ws --save
```
## Usage

### Javascript
```javascript
const mobilyWs = require('mobily-ws');
const client = mobilyWs('username', 'password');
```

### TypeScript
```typescript
import { mobilyWs } from 'mobily-ws';
const client = mobilyWs('username', 'password');

```

## Test
```sh
npm run test
```
